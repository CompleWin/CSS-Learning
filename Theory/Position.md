# Позиционирование CSS

### Статичное: static

Идут по порядку, как указаны в HTML

```css
.element {
    position: static;
}
```

### Относительное: relatice

Идут относительно родительского элемента

```css
.element {
    position: relative;
}
```

### Абсолютное: absolute

Позиционирование идет относительно корневого элемента

```css
.element {
    position: absolute;
}
```

### Фиксированное: fixed

Относительно браузера

```css
.element {
    position: fixed;
}
```
