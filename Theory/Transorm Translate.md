# Transform Translate

```css
.element {
    transform: translateX(100px);
    /* по оси X вправо на 100px */
}
```
```css
.element {
    transform: translateY(-50px);
    /* по оси Y вверх на 50px */
}
```
```css
.element {
    transform: translate(-50%, -50%);
    /* по осям X, Y вверх на половину ширины и высоты */
}
```
